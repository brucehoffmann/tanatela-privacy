# Política de Privacidade — TaNaTela

Última atualização: 2026-04-21

## Quem somos
TaNaTela é um serviço que ajuda estabelecimentos a exibir conteúdos em telas/TVs.

## Quais dados coletamos
- Dados de conta e configuração do estabelecimento (quando aplicável)
- Identificadores de conta profissional do Instagram (ex.: ig-user-id)
- Tokens de acesso do Instagram/Meta (quando aplicável)
- Metadados de conteúdo (ex.: ids, timestamps, links/permalinks)
- Logs técnicos para diagnóstico e segurança

## Como usamos os dados
- Conectar a conta do Instagram/Meta ao TaNaTela
- Buscar metadados necessários para gerar playlists de exibição
- Operar, manter e melhorar o serviço

## Compartilhamento
Não vendemos seus dados. Podemos compartilhar com provedores de infraestrutura (ex.: hospedagem) apenas para operar o serviço.

## Retenção
Mantemos dados pelo tempo necessário para operação e obrigações legais. Conteúdos e logs podem ter retenção limitada conforme configuração.

## Segurança
Aplicamos medidas técnicas e organizacionais para proteger os dados, incluindo controle de acesso e criptografia quando aplicável.

## Seus direitos
Você pode solicitar acesso, correção ou exclusão dos seus dados.
